# Mini Shell Practice

This project is for practicing basic command input and output using Python.  
It simulates a simple shell that can read user commands, handle basic commands, and exit.

## Features
- Read user command
- Simple command handling
- Exit command

## Files
- **test.py** : basic version of mini shell (simple input and output)
- **test2.py** : improved version with command parsing and basic error handling

## Purpose
Learning how a simple shell works and practicing programming logic.

## Status
Learning project / under development

## How to run
```bash
python3 test.py
python3 test2.py

## Example from test.py
